Clarity Starter Instructions:

  • Start by clicking Fork in the toolbar above.
  • Implement the problem in the new editor.
  • Save the result, and share the url in a GitHub or StackOverflow issue.

It helps to ensure you are running the latest version of Clarity, to update you can go to the Dependencies block in the left column, and add @clr/angular@latest @clr/ui@latest @clr/icons@latest to the package input to update Clarity to latest. You can substitute latest for a version number to install a specific version.
